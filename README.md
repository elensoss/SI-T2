## ⚙️ Instalação e Configuração do Ambiente

Antes de executar o projeto, é necessário instalar as bibliotecas de análise de dados e aprendizado de máquina. Certifique-se de ter o [Python](https://www.python.org/downloads/) instalado na sua máquina.

Abra o terminal na pasta do projeto e execute o comando correspondente ao seu sistema operacional:

### Para Windows (Via Terminal / PowerShell)
Utilize o inicializador nativo do Windows para garantir que o pacote seja instalado no ambiente correto:
```powershell
py -m pip install pandas scikit-learn
```
### Para Linux
Utilize o terminal do Linux:
```
python3 -m pip install pandas scikit-learn
```
